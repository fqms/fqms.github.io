What is FQM

Free Queue Manager. Web based management system developed for the purpose of easing the process of organizing queues and lines. Like many other queue management systems, this system does provide a basic control panel that allow the users and customers alike to interact with it via a basic and simple user interface. Point that makes this system different from others, the fact it's free to use and share with others, which makes it a good free alternative. Moreover, The independent nature of this system, has influenced the development towards more independent IT solutions, that won't require you to be more dependant on specific operating system or hardware, instead finding new solutions that fits all, is the main goal.

Major Features

Web-based platform

It gives the ability to multiple users and devices to interact with system at the same time, without the need for Internet access. It has the ability to run on any local network .

Customization

Gives the user the ability to customize every interface , granting a unique and special experience for every user .

User management

Offers a user-based platform, so multiple users have the ability to register, log-in and monitor their own assigned by Admin tasks .

POS Printers

Support for major POS USB printers on the market. By a simple click, you can assign any connected USB printer.

Multimedia

Offers a multimedia platform, that any permitted user can upload videos, images or sound files to, so they can be incorporated in the customization process .

Verbal announcement

Supports text-to-speech clear and high-quality verbal announcement, in two different languages English and Arabic and with different preferences and settings .
